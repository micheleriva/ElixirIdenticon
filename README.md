# Identicon

A simple Identicon generator, based on a great tutorial on Udemy.

![Identicon in action](/docs/identicon.gif)

# Installation

```sh
$ git clone git@github.com:micheleriva/ElixirIdenticon.git
$ cd ElixirIdenticon
$ mix deps.get
$ iex -S mix
```
